Research & Work

1. FaceTrack AI: Intelligent Head Pose Detection for Smart Attendance and Exam Monitoring

This project builds a system that automatically detects and tracks head poses using machine learning and computer vision. By analyzing facial landmarks and head orientation (yaw, pitch, roll), the model determines whether a student is attentive or distracted, and can automate attendance verification during exams.

Main Programming Language: Python (OpenCV, TensorFlow, NumPy)

Applications: Educational monitoring, automatic attendance, and behavior analysis in security contexts.

2. Personal GitHub Portfolio Website

This project is a personal academic web platform designed to present my background, achievements, and projects in a unified format. It demonstrates my skills in HTML and CSS, and serves as an academic showcase for networking and professional communication.

Main Technologies: HTML, CSS

3. SmartFlash: Interactive Flashcard Learning App

A Python-based console application that helps users efficiently learn and memorize information. It allows users to create, view, and quiz themselves with flashcards, supporting randomization and score tracking.

Main Programming Language: Python

Usage: Ideal for studying languages, formulas, or general knowledge. I personally used it for preparing English and Chinese exams.
