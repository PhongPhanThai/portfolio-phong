Projects and Works

1/FaceTrack AI: Intelligent Head Pose Detection for Smart Attendance and Exam Monitoring

This project build a system that can automatically detect and track head poses using machine learning and computer vision. By analyzing facial landmarks and head orientation (yaw, pitch, roll), the model can determine whether a student is looking forward, distracted, or attempting to cheat during exams. The system can also be used to automate roll calls by recognizing faces and confirming presence in real time.

Main Programming Language: Python

(OpenCV, TensorFlow, NumPy libraries)

The system can be applied in education for automatic attendance and cheating detection during exams. Ngoài ra, nó cũng có thể ứng dụng cho các mục đích security and giám sát hành vi.
